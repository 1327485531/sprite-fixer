# Sprite Fixer 🛠️

**一键修复 AI 生成的 4x4 序列图：去黑底 + 稳定帧抖动。**

A tiny tool to fix AI-generated sprite sheets: auto-remove black backgrounds and stabilize frame jitter.

---

## 它能做什么 | What it does

- ✅ **去黑底**：自动识别并移除纯黑背景，支持 Additive 叠加模式
- ✅ **稳抖动**：对齐每一帧的锚点，减少 AI 序列图的帧间抖动
- ✅ **批量处理**：支持导入 4x4 序列图，一键导出干净素材

## 使用方法 | How to use

1. 打开 [工具链接]
2. 导入你的 4x4 序列图
3. 自动检测背景，去掉
4. 重心对齐，包围盒，总有一个能让图片不抖
5. 导出干净的序列图，丢进引擎直接用

---

## 适用场景 | Use cases

- AI 生成的特效序列图（火焰、雷电、爆炸等）
- 独立游戏开发中的临时素材处理
- 任何需要快速去黑底和稳帧的像素动画
